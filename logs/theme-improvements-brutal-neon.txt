BRUTAL THEME IMPROVEMENTS
───────────────────────────────────────────────────────────────────────────────

✅ CONTRAST FIXES:
  - Changed text-secondary from #000000 to #000000 (maintained)
  - Changed text-muted from #666 to #333333 (darker for better contrast)
  - Changed bg-secondary from #000 to #f5f5f5 (light gray for contrast)
  - Changed bg-tertiary from #ff0 to #000000 (black for strong contrast)
  - Ensured all text on white background is pure black (#000000)

✅ ABSTRACT BRUTALIST DESIGN:
  - Added diagonal stripe pattern background (repeating-linear-gradient)
  - Increased border width from 4px to 6px for more impact
  - Added slight rotation to cards (alternating angles)
  - Cards shift on hover with shadow effects
  - Buttons have uppercase text, bold borders, and shift effect
  - Sections and articles have strong borders
  - Abstract geometric feel with rotated elements

✅ VISUAL ENHANCEMENTS:
  - Cards alternate rotation (even cards rotate opposite direction)
  - Hover effects: cards shift and create shadow boxes
  - Buttons transform with shadow effects on hover
  - Strong, unapologetic borders throughout
  - Pattern background adds texture

NEON THEME IMPROVEMENTS
───────────────────────────────────────────────────────────────────────────────

✅ REDUCED BLOOM EFFECTS:
  - Removed excessive text-shadow from all elements
  - Only headings, accents, and buttons have subtle glow
  - Changed from strong glow (30px) to subtle glow (10px)
  - Reduced shadow intensity from 0.8 to 0.4

✅ INTERACTIVE LIGHTING EFFECTS:
  - Cards now have radial gradient lighting on hover
  - Lighting effect expands from center on hover
  - Border glows with neon color on hover
  - Box shadow creates "lighting around the box" effect
  - Inset shadow adds inner glow
  - Text gets subtle glow on hover

✅ BUTTON LIGHTING:
  - Buttons have radial gradient that expands on hover
  - Creates lighting effect from center outward
  - Border glows with neon accent color
  - Background gets subtle tint on hover
  - Text glows on hover

✅ COLOR ADJUSTMENTS:
  - Changed bg-secondary from #161b22 to #0a0a0a (darker)
  - Changed bg-tertiary from #21262d to #1a1a1a (darker)
  - Changed border color to match accent (#00ff88)
  - Improved contrast while maintaining neon aesthetic

TECHNICAL DETAILS
───────────────────────────────────────────────────────────────────────────────

BRUTAL THEME CSS Variables:
  - --border-width: 6px (increased from 4px)
  - --brutal-pattern: diagonal stripe pattern
  - --color-bg-secondary: #f5f5f5 (light gray)
  - --color-bg-tertiary: #000000 (black for contrast)

NEON THEME CSS Variables:
  - --neon-glow-subtle: 0 0 5px, 0 0 10px (reduced)
  - --neon-glow-medium: 0 0 10px, 0 0 20px (for hover)
  - --neon-light-hover: multi-layer glow effect

NEW CSS FEATURES:
  - [data-theme="neon"] .card::before - radial gradient overlay
  - [data-theme="neon"] .btn::before - expanding light effect
  - [data-theme="brutal"] .card rotation - alternating angles
  - [data-theme="brutal"] background-image - pattern texture

TESTING RECOMMENDATIONS
───────────────────────────────────────────────────────────────────────────────

1. Test Brutal theme:
   - Verify text is readable on all backgrounds
   - Check card rotation effects
   - Test button hover effects
   - Verify pattern background is visible

2. Test Neon theme:
   - Hover over cards to see lighting effect
   - Hover over buttons to see radial lighting
   - Verify glow is subtle, not overwhelming
   - Check that text is readable

3. Run visual assessment:
   npm run visual-assessment:advanced

4. Check contrast ratios:
   npm run verify-themes
