✅ BRUTAL THEME - FIXED CONTRAST & ADDED ABSTRACT DESIGN
───────────────────────────────────────────────────────────────────────────────

PROBLEM: Background and text colors were nearly the same, making it hard to read.

FIXES APPLIED:
1. ✅ Improved Contrast:
   - Changed bg-secondary: #000 → #f5f5f5 (light gray for contrast)
   - Changed bg-tertiary: #ff0 → #000000 (black for strong contrast)
   - Changed text-muted: #666 → #333333 (darker for better readability)
   - Ensured all text on white is pure black (#000000)

2. ✅ Added Abstract Brutalist Design:
   - Diagonal stripe pattern background (repeating-linear-gradient at 45deg)
   - Increased border width from 4px to 6px
   - Cards have slight rotation (alternating angles)
   - Cards shift and create shadow boxes on hover
   - Buttons have uppercase text, bold borders, shift effect
   - Sections and articles have strong 6px borders
   - Abstract geometric feel with rotated elements

3. ✅ Visual Enhancements:
   - Cards alternate rotation (even cards rotate opposite)
   - Hover: cards shift with shadow box effect
   - Buttons transform with shadow on hover
   - Strong, unapologetic borders throughout
   - Pattern background adds texture

✅ NEON THEME - REDUCED BLOOM & ADDED INTERACTIVE LIGHTING
───────────────────────────────────────────────────────────────────────────────

PROBLEM: Bloom effects were too much, needed interactive lighting on hover.

FIXES APPLIED:
1. ✅ Reduced Bloom Effects:
   - Removed excessive text-shadow from all elements
   - Only headings, accents, and buttons have subtle glow
   - Changed from strong glow (30px) to subtle glow (10px max)
   - Reduced shadow intensity from 0.8 to 0.4
   - Glow only on specific elements, not everything

2. ✅ Added Interactive Lighting Effects:
   - Cards: Radial gradient lighting expands from center on hover
   - Cards: Border glows with neon color (#00ff88) on hover
   - Cards: Multi-layer box shadow creates "lighting around the box"
   - Cards: Inset shadow adds inner glow effect
   - Cards: Text gets subtle glow on hover
   - Buttons: Radial gradient expands from center on hover
   - Buttons: Creates lighting effect from center outward
   - Buttons: Border glows with neon accent color
   - Buttons: Background gets subtle tint on hover

3. ✅ Color Adjustments:
   - Changed bg-secondary: #161b22 → #0a0a0a (darker)
   - Changed bg-tertiary: #21262d → #1a1a1a (darker)
   - Changed border color to match accent (#00ff88)
   - Improved contrast while maintaining neon aesthetic

HOW TO TEST
───────────────────────────────────────────────────────────────────────────────

1. Visit the live site: https://joyful-cranachan-6fdbb1.netlify.app

2. Switch to Brutal theme:

3. Switch to Neon theme:

   - Verify glow is subtle, not overwhelming
   - Check that text is readable

4. Run visual assessment:
   npm run visual-assessment:advanced
